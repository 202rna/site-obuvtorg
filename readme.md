## 🛠 Спецификация API Маршрутов

Все эндпоинты разделены по типам HTTP-методов. Полную интерактивную документацию (Swagger) можно посмотреть локально после запуска по адресу: http://localhost:8000/docs#/

# API Documentation

Спецификация и список доступных эндпоинтов для взаимодействия с API. 

🔒 — метод требует авторизации (передачи токена в заголовках).

## Аутентификация и пользователи

| Метод | Эндпоинт | Защита | Описание |
| :--- | :--- | :---: | :--- |
| `POST` | `/api/register` |  | Регистрация нового пользователя |
| `POST` | `/api/login` |  | Авторизация пользователя (вход) |
| `GET` | `/api/me` | 🔒 | Получение информации о текущем пользователе |

## Товары (Products)

| Метод | Эндпоинт | Защита | Описание |
| :--- | :--- | :---: | :--- |
| `GET` | `/api/products` |  | Получение списка всех товаров |
| `POST` | `/api/products` | 🔒 | Добавление нового товара |
| `GET` | `/api/products/{product_id}` |  | Получение информации о товаре по его ID |
| `DELETE` | `/api/products/{product_id}` | 🔒 | Удаление товара по его ID |

## Корзина (Cart)

| Метод | Эндпоинт | Защита | Описание |
| :--- | :--- | :---: | :--- |
| `GET` | `/api/cart` | 🔒 | Получение содержимого корзины |
| `DELETE` | `/api/cart` | 🔒 | Очистка всей корзины |
| `POST` | `/api/cart/{product_id}` | 🔒 | Добавление товара в корзину |

## Заметки (Notes)

| Метод | Эндпоинт | Защита | Описание |
| :--- | :--- | :---: | :--- |
| `POST` | `/api/note` | 🔒 | Создание новой заметки |
| `DELETE` | `/api/note/{note_id}` | 🔒 | Удаление заметки по её ID |
| `PATCH` | `/api/note/{note_id}` | 🔒 | Частичное обновление заметки по её ID |
| `GET` | `/api/note/{note_id}` |  | Получение одной конкретной заметки по её ID |
| `GET` | `/api/notes` |  | Получение списка всех заметок |

## 🐳 Быстрый запуск (Docker)

Проект полностью контейнеризирован. Для сборки образов и запуска всех необходимых сервисов (FastAPI, СУБД) в фоновом режиме выполните следующую команду в корневой папке проекта:

```bash
docker compose up --build -d
```

Для остановки контейнеров используйте:

```bash
docker compose down
```
